Все оболочки типов переопределяют метод toString(), который возвращает удобочитаемую форму значения
ручная распаковка: int i = iOb.intValue();

Автоупаковка: Integer iOb = 100; Автораспаковка: int i = iOb;
допустимы такие действия как: ++iOb; iOb += 10; iOb2 = iOb + (iOb / 3); i = iOb + (iOb / 3); (switch case тоже)
