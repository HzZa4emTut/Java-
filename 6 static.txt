P.S. Статический вложенный класс имеет прямой доступ только к остальным статическим членам
основного класса. Доступ к другим членам его внешнего класса должен осуществляться через объектную ссылку.


Статическое импортирование
Ключевое слово import вместе со static можно применять для импортирования статических членов класса или интерфейса
вместо Math.sqrt(4), после import static java.lang.Math.sqrt, будет sqrt(4)
также можно импротировать все статические члены заданного класса или интерфейса: import static пакет.имя-типа.*;
